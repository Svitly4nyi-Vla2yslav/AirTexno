# AirTexno Website

React-based website for AirTexno appliance repair services.

## Tech Stack
- React 19 + TypeScript + Vite
- Material-UI + styled-components
- Multilingual support (en/es/ru/uk)
- Deployed on Netlify

## Development
```bash
npm install
npm run dev
```

## Deployment
Automatically deploys to Netlify on push to main branch.
